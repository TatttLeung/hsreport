## 合作文档

### 栏目

#### 人员信息管理：

1 学生信息 ：学号，姓名，性别，年级 ，课程编码，课程名称
2 教师信息：编号，姓名，性别 
3 教师任课信息：教师姓名，所教课程 ，上课时间
4 课程学生信息：学生名单
5 课程导入学生信息 ：课程，上课时间，教师，学生信息（打勾）

#### 课程材料提交：

1 课程信息设置：编号，教师姓名，课程名称，学年，学期，上课时间，作业提交次数
2 学生材料提交：学号、姓名、课程名称、作业编号、文件提交
3 教师材料审核：
4 材料审核进度查询
